**2.应用背景** 

进行二维数组的存储，其中有大量的重复的值为0的数据。

如果按原数据结构构建为6 * 7的二维数组，则过于浪费空间，可以将为0的数据省略。

具体代码见java文件

**3.小结**

稀疏数组主要目的为节省空间，用于记录有大量重复值的二维数组。

当一个数组中大部分元素为0，或者为同一值的数组时，可以使用稀疏数组来保存该数组。

稀疏数组的处理方式是:记录数组一共有几行几列，有多少个不同值；把具有不同值的元素和行列及值记录在一个小规模的数组中，从而缩小程序的规模。
